# luogu
## 你需要什么
1. python3 (推荐python3.13)
2. pip
3. requests
4. beautifulsoup4
5. html2text
## 安装
``` sh
pip install requests beautifulsoup4 html2text
```
## 注意事项
1. 请不要频繁爬虫，否则可能会被封禁。
2. 请遵守 `robots.txt` 协议和`luogu`的用户协议。
3. 请使用`edge`或`firefox`浏览器，其余浏览器获得的`cookie`可能无法使用。
